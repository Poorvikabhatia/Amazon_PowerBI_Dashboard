# Amazon Product Analysis Dashboard 📊

## 📌 Project Overview

This project presents an interactive Power BI dashboard analyzing Amazon product data, including pricing trends, customer ratings, and category-wise performance.

## 📂 Dataset

* 1465+ products
* Features include price, rating, discount, category, and reviews

## ⚙️ Tools Used

* Power BI
* Python (Pandas for data cleaning)

## 📊 Dashboard Insights

* Product distribution across categories
* Revenue contribution by category
* Rating trends across price ranges
* Top-performing products
* Rating distribution

## 🚀 Key Features

* Interactive slicers for filtering
* KPI cards for quick insights
* Clean and professional UI design

## 📁 Files Included

* `Amazon Dashboard.pbix`
* `amazon_cleaned.csv`
* `Amazon_Dashboard.pdf`

## 💡 Conclusion

This dashboard helps understand customer preferences, pricing strategies, and product performance on Amazon.
